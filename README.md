# EncubatorrApp
## Overview
EncubatorrApp is an AI-powered business builder incubator platform designed to enable users to build, manage and launch a new business, step-by-step from idea to launch.
## Tech Stack
- Frontend: React / Next.js
- Backend: Node.js / Express
- Database: PostgreSQL
- Hosting: Vercel or AWS
- Version Control: GitHub
## Folder Structure
/encubatorr-app/
│
├── /client/       # Frontend code (React or Next.js)
├── /server/       # Backend API logic
├── /docs/         # Product specs and documentation
├── /scripts/      # Setup or utility scripts
├── /tests/        # Unit and integration tests
├── README.md      # Project intro and setup instructions
├── .env.example   # Template for environment variables

Getting Started (Local Setup)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/EncubatorrOrg/encubatorr-app.git

2. Install dependencies:
For frontend:
cd client
npm install
For backend:
cd ../server
npm install
(Repeat for `/server` if needed)
4. Run the app locally:
For Frontend
npm run dev
For Backend
npm run start
